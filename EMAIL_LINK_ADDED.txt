CONFIGURATION
-------------

To change the URL (for production):

Edit config.py:
frontend_url: str = "https://samwylock.com"  # Change this

Or set environment variable:
FRONTEND_URL=https://samwylock.com

The email service automatically uses settings.frontend_url
