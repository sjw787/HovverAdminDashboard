HOW FRONTEND USES IT
---------------------

Simple Logic:
```javascript
const showResendButton =
  customer.user_status === 'FORCE_CHANGE_PASSWORD' ||
  customer.user_status === 'RESET_REQUIRED';
```

That's it! If true, show the "Resend Welcome Email" button.
If false, hide it.

THE TWO KEY STATUSES
--------------------

‚úÖ FORCE_CHANGE_PASSWORD
   - Customer hasn't logged in yet
   - Or logged in but didn't complete password change
   - SHOW resend button

‚ùå CONFIRMED
   - Customer has completed initial login
   - Customer set their own password
   - HIDE resend button
   - They should use "Forgot Password" instead
